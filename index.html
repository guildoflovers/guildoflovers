Власний сервер

В нашій гільдії є сервер з модами, серед яких Create та Ars Nouveau. Create додає механічні елементи для автоматизації, а Ars Nouveau — магічні закляття та предмети. Ці моди відкривають нові можливості для створення складних конструкцій та магічних систем.

Власна збірка модів

В нашій гільдії є власна збірка модів, яка включає найкращі доповнення для Minecraft. Ми обрали моди, які додають нові механіки, можливості для автоматизації та магії, щоб кожен гравець міг реалізувати свої ідеї та насолоджуватися грою на новому рівні.
